# Chrome-Extension

How to Install this extesion:
```
1). Go to chrome://extensions/ and check the box for Developer mode in the top right.
2). Download Developer Shortcut.zip
3). Locate the ZIP file on your computer and unzip it.
3). Go back to the chrome://extensions/ page and click the Load unpacked extension
    button and select the unzipped folder for your extension to install it.
```
